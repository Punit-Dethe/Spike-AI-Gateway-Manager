# Assets Folder

This folder contains application assets for NexusAI Gateway.

## Required Files

### icon.ico
**Required for building the installer**

- **Format**: Windows Icon (.ico)
- **Sizes**: 256x256, 128x128, 64x64, 48x48, 32x32, 16x16
- **Purpose**: Application icon for installer, shortcuts, and taskbar

**How to create**: See [../ICON_GUIDE.md](../ICON_GUIDE.md)

## Before Building

Make sure `icon.ico` exists in this folder before running `build.bat`.

If you don't have an icon yet:
1. See [ICON_GUIDE.md](../ICON_GUIDE.md) for creation instructions
2. Use an online converter to create a placeholder
3. Or hire a designer for a professional icon

## Placeholder Icon

For testing purposes, you can create a simple placeholder:

1. Create a 512x512 PNG with text "NA"
2. Use an online ICO converter
3. Save as `icon.ico` in this folder

**Online converters:**
- https://www.icoconverter.com/
- https://convertio.co/png-ico/
- https://cloudconvert.com/png-to-ico
